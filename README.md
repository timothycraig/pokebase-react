# Pokebase React

### Search for your favorite pokemon and look up their stats

![Screenshot](http://imgur.com/a/8JWeh)

##### Boot up the API server
```
$ cd pokebase-react
$ cd api-server
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

##### Boot up the client server
```
$ ..
$ npm install
$ npm start
$ open http://localhost:8080/
```
